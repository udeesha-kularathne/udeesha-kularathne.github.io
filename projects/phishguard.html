PhishGuard – Intelligent Phishing Detection System

A real-time phishing detection platform that evaluates messages and links using hybrid machine learning, anomaly detection, and explainable AI, delivered directly through a browser extension and low-latency inference APIs.

Role ML Engineer & Full-Stack Developer
Timeline 2023 - 2024
Category AI/ML, Cybersecurity

Project Overview

PhishGuard was developed to address the limitations of single-model phishing detection systems in real-world environments. The project focuses on combining multiple analytical signals into a unified decision pipeline that can operate in real time while remaining transparent and interpretable.

The system follows a modular design where offline model training, online inference, and user-facing interaction are clearly separated. Machine learning models are trained and evaluated independently, packaged for inference, and exposed through a lightweight Flask API. A browser extension consumes these APIs to deliver immediate risk feedback as users view messages or links.

Special emphasis is placed on explainability and robustness. In addition to classification outputs, the system provides feature-level insights and attribution signals that help explain why a message is flagged, supporting trust and auditability in security-sensitive contexts.

Note: Model thresholds, training data, and security heuristics are abstracted to preserve confidentiality and operational safety.

Key Features

Hybrid Phishing Classification

Combines transformer-based language understanding with traditional feature-driven classifiers to reduce dependence on a single detection signal.

Anomaly Detection Layer

Identifies structurally unusual or previously unseen message patterns that may bypass standard supervised models.

Explainable AI Outputs

Highlights influential words, patterns, and features using attribution methods and rule-based explanations to justify predictions.

Browser Extension Integration

Delivers phishing risk analysis directly within the browsing experience, enabling immediate feedback without disrupting user workflows.

Low-Latency Inference API

A lightweight Flask-based inference service optimized for fast startup and consistent response times in real-time environments.

Technology & Engineering Stack

Python Flask REST APIs CORS Transformers scikit-learn XGBoost Anomaly Detection Model Stacking Explainable AI Browser Extensions JavaScript API-Driven Design

Results & Engineering Impact

PhishGuard demonstrates how layered machine learning strategies can improve robustness against evolving phishing techniques. By combining supervised classification, anomaly detection, and trust signals, the system reduces single-model failure modes while maintaining consistent inference behavior.

From an engineering perspective, the project emphasizes clean separation between training, inference, and user interaction layers. This design allows models to be retrained or replaced without requiring changes to the browser extension or frontend logic, supporting long-term maintainability and secure iteration.
